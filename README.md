# Love Letter Website

A simple interactive website with an openable letter that reveals "Je t'aime" (I love you in French).

## How to View

1. Open `index.html` in your web browser.
2. Click on the envelope to open the letter.

## Files

- `index.html`: The main HTML structure.
- `styles.css`: Styling for the envelope and letter.
- `script.js`: JavaScript to handle the opening animation.

Enjoy!
